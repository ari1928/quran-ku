# Quran'Ku
Al Qur'an web application with progressive web apps

## Getting Started

Run development server:

```shell
$ npm run dev
or
$ yarn dev
```

Build project:

```shell
$ npm run build
or
$ yarn build
```

Serve project on localhost:

```shell
$ npm start
or
$ yarn start
```

Open http://localhost:3000 in your local machine
